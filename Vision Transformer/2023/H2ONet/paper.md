# H2ONet: Hand-Occlusion-and-Orientation-Aware Network for Real-Time 3D Hand Mesh Reconstruction

---


## Abstract

실시간 3D 손 메시 재구성은 특히 손이 물체를 잡고 있을 때 까다로움

H2ONet
- 여러 프레임에서 가려지지 않은 정보를 최대한 활용하여 재구성 품질을 높이도록 설계
- 먼저, 손 메시 정찰 구조를 두 갈래로 분리
    - 하나는 손가락 수준의 가려지지 않은 정보를 활용하기 위한 것
    - 다른 하나는 실시간 추론을 촉진하기 위해 경량 구조를 사용하여 전역 손 방향을 활용하기 위한 것
- 둘째, 손가락 수준 오클루전 인식 기능 융합을 제안
    - 예측된 손가락 수준 occlusion 정보를 활용하여 시간 프레임에 걸쳐 손가락 수준 정보를 융합하기 위한 지침을 제공
- 핸드 레벨의 오클루전 인식 기능 융합을 설계하여 근처의 시간 프레임에서 occluded 되지 않은 정보를 가져옴

- 도전적인 손-물체 폐색 사례로 Dex-YCB 및 HO3D-v2 데이터 세트에 대한 실험을 수행
    - H2ONet이 실시간으로 실행될 수 있음
    - 손 메시와 포즈 정밀도 모두에서 SOTA 성능을 달성한다는 것을 입증
